= 警报参考
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
此参考列出了网格管理器中出现的默认警报。您收到的警报消息中包含建议的操作。

根据需要，您可以创建自定义警报规则以适合您的系统管理方法。

一些默认警报使用link:commonly-used-prometheus-metrics.html["Prometheus 指标"]。
