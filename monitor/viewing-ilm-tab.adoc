= 查看 ILM 选项卡
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ILM 选项卡提供有关信息生命周期管理 (ILM) 操作的信息。

每个存储节点、每个站点和整个网格都会显示 ILM 选项卡。对于每个站点和网格，ILM 选项卡显示 ILM 队列随时间变化的图表。对于网格，此选项卡还提供完成所有对象的完整 ILM 扫描的预计时间。

对于存储节点，ILM 选项卡提供有关擦除编码对象的 ILM 评估和后台验证的详细信息。
