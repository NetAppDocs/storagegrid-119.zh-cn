= 监控什么以及何时监控
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
即使StorageGRID系统在发生错误或网格部分不可用时可以继续运行，您也应该在潜在问题影响网格的效率或可用性之前对其进行监控和解决。

.开始之前
* 您已使用link:../admin/web-browser-requirements.html["支持的网络浏览器"]。
* 你有link:../admin/admin-group-permissions.html["特定访问权限"]。
