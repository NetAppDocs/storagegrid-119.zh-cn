= 所有三个网络的拓扑
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以将所有三个网络配置为由私有网格网络、有界站点特定管理网络和开放客户端网络组成的网络拓扑。如果需要，使用负载平衡器端点和不受信任的客户端网络可以提供额外的安全性。

在此示例中：

* 网格网络用于与内部对象管理操作相关的网络流量。
* 管理网络用于与管理功能相关的流量。
* 客户端网络用于与 S3 客户端请求相关的流量。
