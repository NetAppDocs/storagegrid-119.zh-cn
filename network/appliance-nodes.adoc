= 设备节点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以配置StorageGRID设备上的网络端口，以使用满足吞吐量、冗余和故障转移要求的端口绑定模式。

StorageGRID设备上的 10/25-GbE 端口可以配置为固定或聚合绑定模式，以连接到网格网络和客户端网络。

1-GbE 管理网络端口可以配置为独立或主动备份模式，以连接到管理网络。
