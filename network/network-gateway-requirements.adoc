= 网络网关要求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
请遵循网络网关或路由器的这些要求和指南。

* 如果设置，给定网络的网关必须位于特定网络的子网内。
* 如果使用静态寻址配置接口，则必须指定除 0.0.0.0 以外的网关地址。
* 如果您没有网关，最佳做法是将网关地址设置为网络接口的 IP 地址。
