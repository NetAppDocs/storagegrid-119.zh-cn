= 客户端请求冲突
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
冲突的客户端请求（例如两个客户端写入同一个密钥）将根据“最新胜利”的原则解决。

“最新胜利”评估的时间取决于StorageGRID系统完成给定请求的时间，而不是 S3 客户端开始操作的时间。
