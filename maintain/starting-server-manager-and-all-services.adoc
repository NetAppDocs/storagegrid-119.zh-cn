= 启动服务器管理器和所有服务
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可能需要启动服务器管理器，它也会启动网格节点上的所有服务。

.开始之前
你有 `Passwords.txt`文件。

.关于此任务
在已运行服务器管理器的网格节点上启动服务器管理器将导致服务器管理器和网格节点上的所有服务重新启动。

.步骤
. 登录网格节点：
+
.. 输入以下命令： `ssh admin@_grid_node_IP_`
.. 输入 `Passwords.txt`文件。
.. 输入以下命令切换到root： `su -`
.. 输入 `Passwords.txt`文件。


+
当您以 root 身份登录时，提示符将从 `$`到 `#`。

. 启动服务器管理器： `service servermanager start`
. 退出命令外壳： `exit`
