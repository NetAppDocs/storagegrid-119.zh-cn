= 重新启动服务器管理器和所有服务
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可能需要重新启动服务器管理器和网格节点上运行的所有服务。

.开始之前
你有 `Passwords.txt`文件。

.步骤
. 登录网格节点：
+
.. 输入以下命令： `ssh admin@_grid_node_IP_`
.. 输入 `Passwords.txt`文件。
.. 输入以下命令切换到root： `su -`
.. 输入 `Passwords.txt`文件。


+
当您以 root 身份登录时，提示符将从 `$`到 `#`。

. 重新启动服务器管理器和网格节点上的所有服务： `service servermanager restart`
+
服务器管理器和网格节点上的所有服务都停止然后重新启动。

+

NOTE: 使用 `restart`命令与使用 `stop`命令后跟 `start`命令。

. 退出命令外壳： `exit`
