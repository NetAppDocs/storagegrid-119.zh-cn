= 停止服务器管理器和所有服务
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
服务器管理器旨在始终运行，但您可能需要停止服务器管理器和网格节点上运行的所有服务。

.开始之前
你有 `Passwords.txt`文件。

.步骤
. 登录网格节点：
+
.. 输入以下命令： `ssh admin@_grid_node_IP_`
.. 输入 `Passwords.txt`文件。
.. 输入以下命令切换到root： `su -`
.. 输入 `Passwords.txt`文件。


+
当您以 root 身份登录时，提示符将从 `$`到 `#`。

. 停止服务器管理器和网格节点上运行的所有服务： `service servermanager stop`
+
服务器管理器和网格节点上运行的所有服务都将正常终止。服务最多可能需要 15 分钟才能关闭。

. 退出命令外壳： `exit`
