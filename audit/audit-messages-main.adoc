= 审计消息描述
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以下章节列出了系统返回的审计消息的详细说明。每个审计消息首先列在一个表中，该表根据消息所代表的活动类别对相关消息进行分组。这些分组对于理解所审计的活动类型以及选择所需的审计消息过滤类型都很有用。

审计消息也按四字符代码的字母顺序列出。此按字母顺序排列的列表可让您查找有关特定消息的信息。

本章中使用的四字符代码是审计消息中的 ATYP 值，如以下示例消息所示：

[listing, subs="specialcharacters,quotes"]
----
2014-07-17T03:50:47.484627
\[AUDT:[RSLT(FC32):VRGN][AVER(UI32):10][ATIM(UI64):1405569047484627][*ATYP\(FC32\):SYSU*][ANID(UI32):11627225][AMID(FC32):ARNI][ATID(UI64):9445736326500603516]]
----
有关设置审计消息级别、更改日志目标以及使用外部系统日志服务器存储审计信息的信息，请参阅link:../monitor/configure-audit-messages.html["配置审计消息和日志目标"]
