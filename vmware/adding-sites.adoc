= 添加站点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
安装StorageGRID时，必须创建至少一个站点。您可以创建额外的站点来提高StorageGRID系统的可靠性和存储容量。

.步骤
. 在站点页面上，输入*站点名称*。
. 要添加其他站点，请单击最后一个站点条目旁边的加号，然后在新的*站点名称*文本框中输入名称。
+
根据电网拓扑的需要添加尽可能多的附加站点。您最多可以添加 16 个站点。

+
image::../media/3_gmi_installer_sites_page.gif[本图四周的文字提供了相关说明。]

. 单击“下一步”。
